# Specification

## Summary
**Goal:** Add real food images for three Nellore cuisine dishes (Chepala Pulusu, Prawns Biriyani, and Natu Kodi Kura) in the Cuisine section.

**Planned changes:**
- Generate images for Chepala Pulusu, Prawns Biriyani, and Natu Kodi Kura
- Update `CuisineSection.tsx` to map each of the three dish names to their corresponding generated image files and display them as card banner images
- Keep emoji fallback for any dish without a generated image
- Retain existing South Indian terracotta/gold/teal theme and card layout

**User-visible outcome:** The Chepala Pulusu, Prawns Biriyani, and Natu Kodi Kura cuisine cards now display real food photography as their banner images instead of emoji icons.
